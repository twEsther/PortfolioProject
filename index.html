Hi, I'm
Esther Huang.

1992 年生於台灣台北，朋友笑稱為半個理工腦與半個文組腦的混合體。

畢業於中國文化大學資訊傳播系（前身為印刷傳播學系），也是在這個時候開始培養對於美感的認知，對於影音剪輯、攝影學、色彩學、認知心理學都有一定程度的涉略，並學習使用 Adobe 關於影像處理的軟體（llustrator、Lightroom、Premiere 等）。

高中為理組，相信世間萬物都自有定律，集感性與理性於一身，因緣際會下，於 2020 年六月報名了台北恆逸的 Java 全方位 Web 養成班學習全端網站技術，正式開始了我的工程師之路。並在 Udemy 自學網頁開發，重新掌握架設網站的脈絡，並學習使用 Figma 設計網站架構。
